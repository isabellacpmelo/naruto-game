## How to use

### Project setup
```
pnpm i
```
```
pnpm update
```
```
pnpm upgrade
```
### Compiles and hot-reloads for development
```
pnpm dev
```
or
```
pnpm dev --port=5173
```
### Compiles and minifies for production
```
pnpm build
```
or
```
pnpm build --mode [environment]
```
and
```
pnpm preview
```
### Lints and fixes files
```
pnpm lint:fix
```
